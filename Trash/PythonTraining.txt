1. Giới thiệu về python
Python là gì, giới thiệu chung về python.
Một vài đặc trưng của python và so với các ngôn ngữ khác.
- script language.
- interpreted language.
- dynamic type and strong type language.
- standardized by an international (or national) standard.
- cpython is open-source.
Strong typing means that the type of a value doesn't change in unexpected ways. A string containing only digits doesn't magically become a number, as may happen in Perl. Every change of type requires an explicit conversion.
Dynamic typing means that runtime objects (values) have a type, as opposed to static typing where variables have a type.
